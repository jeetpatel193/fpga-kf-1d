This update delivers a runnable Python golden model of a 1-D Kalman Filter (predict/update). It generates synthetic inputs (noisy constant and ramp), computes filtered estimates, and saves both the inputs and golden outputs to CSV. Plots are included for visual verification.

How to run:
1) python3 -m venv .venv && source .venv/bin/activate
2) pip install -r requirements.txt
3) python golden/golden_kf.py --Q 1e-4 --R 1e-2 --x0 0.0 --P0 1.0

- CSVs in data/: z_*.csv, x_out_*.csv, P_out_*.csv
