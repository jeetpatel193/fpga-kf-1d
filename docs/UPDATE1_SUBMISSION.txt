Project Update 1 — FPGA-Accelerated 1D Kalman Filter (IMU only)

Brief description
A runnable Python golden model of a 1-D Kalman Filter for IMU sensor fusion (single-axis angle).
Model: A=1, B=Δt, C=1. Predict uses gyro rate (input u_k), Update uses accelerometer angle (measurement z_k).
I provide two scenarios (sine and step), write golden outputs (x_hat, P, K) to CSV, generate plots, and include a
scripted checker to compare future HLS outputs against the golden arrays.

What I implemented
• Data generator (golden/gen_imu.py): creates t, x_true, u_gyro (rad/s), z_accel (rad) + plots.
• 1-D KF runner (golden/kf_imu_from_csv.py): computes x_hat, P, K + plots and RMSE vs ground truth.
• Auto-checker (golden/compare_golden.py): MAE/RMSE/MAX and PASS/FAIL vs golden.
